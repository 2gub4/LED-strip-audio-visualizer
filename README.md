# LED-strip-audio-visualizer
Oprogramowana taśma LED WS2812B, ESP32, C++, biblioteki FastLED i arduinoFFT, Arduino IoT Cloud, 20 trybów świecenia w tym specjalne zależne od odległości i częstotliwości dźwięku (FFT), dynamiczne sterowanie efektami świetlnymi, kolorem, jasnością i trybami pracy, zapis ustawień w pamięci EEPROM.
